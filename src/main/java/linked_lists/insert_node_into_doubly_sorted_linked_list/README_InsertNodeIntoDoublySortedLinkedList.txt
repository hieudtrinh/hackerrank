Given a reference to the head of a doubly-linked list and an integer, data,
create a new DoublyLinkedListNode object having data value 'data' and insert
it at the proper location to maintain the sort.

Example
Sample Input

1->3->4->10

data=5
Sample Output

1->3->4->5->10
